# YouTube to Gemini

YouTubeの各動画サムネイルから、動画URLと要約指示をGeminiへ送るChrome拡張です。拡張のUIはChromeの表示言語に合わせて表示されます。

> 非公式の拡張機能です。Google、YouTube、Geminiによる承認・提供を受けた製品ではありません。

## インストール

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパー モード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. このリポジトリのフォルダを選ぶ
5. YouTubeを再読み込みする

動画のサムネイル右下にある「要約」ボタンを押すと、新しいタブでGeminiが開き、要約指示を入力します。内容を確認してGeminiの送信ボタンを押してください。Geminiの画面変更などで自動入力できない場合は、プロンプトをGemini画面上の読み取り専用欄へ表示します。拡張機能がクリップボードへ自動コピーすることはありません。

> 更新後は `chrome://extensions` にある本拡張の再読み込みボタンを押し、YouTubeのタブも再読み込みしてください。Geminiへのアクセス権限を求められた場合は許可します。

## 設定

拡張機能アイコンから次の項目を変更できます。

- Geminiの要約言語をChromeの表示言語に合わせる、または手動で指定
- 標準プロンプト、またはオリジナルプロンプト
- カスタムプロンプト内で `{language}`、`{title}`、`{url}` を展開
- 明示的なオプトインでGeminiの送信ボタンを自動押下

自動送信は初期状態では無効です。有効にすると「要約」ボタンを押した動画のタイトル、URL、設定したプロンプトがGeminiへ自動送信されます。この機能はGeminiの画面構造に依存します。無効の場合は、プロンプトの内容を確認してから手動で送信します。

拡張の設定画面はChromeの表示言語に、YouTube上へ注入するボタンはYouTubeの表示言語に追従します。日本語、英語、簡体字中国語、繁体字中国語、スペイン語、ドイツ語、フランス語に対応し、それ以外では英語になります。

## 対応範囲

- YouTubeホーム
- 検索結果
- 関連動画
- チャンネルの動画一覧
- 再生リスト

YouTubeのDOM変更によりボタンが表示されなくなった場合は、`content.js` のカード・サムネイル用セレクタを調整してください。

## テスト

ChromeまたはChromiumがインストールされた環境で、次を実行します。

```bash
bash scripts/test.sh
```

Manifestと翻訳ファイルの整合性に加え、実ブラウザ上でYouTubeへのボタン注入、プロンプト生成、Geminiへの入力、自動送信を確認します。GitHubへpushした場合もGitHub Actionsで同じテストが実行されます。

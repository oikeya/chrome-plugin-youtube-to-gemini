# Ficha de la tienda — Español

## Título

YouTube to Gemini

## Resumen

Envía vídeos de YouTube a Gemini para resumirlos.

## Categoría

Productividad

## Descripción detallada

Añade un botón Resumir a las miniaturas de YouTube y prepara en Gemini el título, la URL y el prompt del vídeo seleccionado.

- Funciona en inicio, búsquedas, canales, vídeos relacionados y listas.
- El idioma del resumen sigue al navegador de forma predeterminada y puede elegirse manualmente.
- Prompts personalizados con `{language}`, `{title}` y `{url}`.
- De forma predeterminada, permite revisar el prompt antes de enviarlo.
- El envío automático está desactivado y requiere activación explícita.

El desarrollador no opera ningún servidor ni recibe datos analíticos o de navegación. El título, la URL y el prompt solo se envían directamente a Google Gemini después de pulsar Resumir.

Extensión no oficial, no afiliada, respaldada ni proporcionada por Google, YouTube o Gemini.

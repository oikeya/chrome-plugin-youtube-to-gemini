# Store assets

Chrome Web Store申請専用の素材です。拡張ZIPには含めません。

## 用意済み

- `promo-small-440x280.png`: 必須の小型プロモ画像
- `promo-marquee-1400x560.png`: 任意のマルキー画像
- `listing/`: ローカライズした掲載文
- `REVIEWER_NOTES.md`: 審査説明、権限理由、データ申告、テスト手順

## ユーザーが撮影するもの

実際のYouTube/Geminiアカウント状態を必要とするため、[screenshots/README.md](screenshots/README.md)を確認して撮影してください。スクリーンショットは実際の利用体験を示し、個人情報や視聴履歴が映らないようにします。

必要寸法：`1280x800`または`640x400`、PNG/JPEG、最低1枚。

YouTube to Gemini

An unofficial Chrome extension. Not affiliated with, endorsed by, or provided by Google, YouTube, or Gemini.

Add a summary button to YouTube video thumbnails and prepare the selected video's title, URL, and prompt in Gemini.

YouTubeのサムネイルへ要約ボタンを追加し、選択した動画のタイトル、URL、プロンプトをGeminiへ渡す非公式Chrome拡張です。

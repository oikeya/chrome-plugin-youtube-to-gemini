Privacy Policy / プライバシーポリシー

Effective date / 施行日: 2026-06-21

English

YouTube to Gemini handles the title and URL of the YouTube video whose Summarize button the user clicks, the user's summary settings and custom prompt, and the final assembled prompt.

Settings are stored with Chrome Storage Sync. The final prompt is temporarily stored in Chrome local extension storage, copied to the clipboard as a fallback, and disclosed directly to Google Gemini solely to provide the summary requested by the user. Automatic sending occurs only after the user explicitly enables that option.

The developer operates no server for this extension and does not receive, retain, sell, or use this data for advertising. The extension does not collect credentials, cookies, complete browsing history, analytics identifiers, or payment data. Google services process data under Google's own terms and privacy policy.

Settings remain until changed, cleared, or the extension is uninstalled. Pending prompts are removed after use or expiration. Clipboard content remains until overwritten.

The use of information received from Chrome APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.

日本語

YouTube to Geminiは、ユーザーが要約ボタンを押したYouTube動画のタイトルとURL、要約設定、カスタムプロンプト、完成したプロンプトを取り扱います。

設定はChrome Storage Syncへ保存します。完成したプロンプトはChromeの拡張機能用ローカルストレージへ一時保存し、失敗時に備えてクリップボードへコピーしたうえで、ユーザーが要求した要約を生成する目的に限りGoogle Geminiへ直接渡します。自動送信は、ユーザーが設定で明示的に有効化した場合のみ行います。

開発者は本拡張用のサーバーを運用しておらず、データを受信、保持、販売、広告利用しません。認証情報、Cookie、閲覧履歴全体、アクセス解析識別子、決済情報は収集しません。Googleサービス上の処理にはGoogleの規約とプライバシーポリシーが適用されます。

設定は変更、消去、またはアンインストールまで保存されます。処理待ちプロンプトは使用後または有効期限後に削除され、クリップボードは上書きまで残ります。

Chrome APIから取得した情報の利用は、Limited Use要件を含むChrome Web Store User Data Policyに準拠します。
